# Password Strength Checker (Python)

A Python-based password strength checker that evaluates password security and provides improvement suggestions using regex-based validation.

## Features

Checks password length

Validates uppercase & lowercase letters

Ensures numbers and special characters

Detects weak/common passwords

Provides improvement suggestions

Secure password input using getpass

## Tools Used

Python 3

Regular Expressions (re)

getpass for hidden password input

## How to Run

1.Clone the repository:

git clone https://github.com/your-username/password-strength-checker.git


2.Navigate to the project folder:

cd password-strength-checker


3.Run the script:

python password_checker.py


4.Enter your password when prompted.

Example:
Enter password to check strength: ************
